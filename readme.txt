== Description ==

Adds a simple author box with Gravatar image below the articles on single page. You can add Twitter and Facebook profiles.

== Installation ==

Upload the plugin to wp-content/plugins or install it using the automatic installer. 
Activate the plugin. Once the plugin is activated it will delete YM and AIM fields in your profile and add fields for Twitter and Facebook profiles. The plugin checks if the fields are empty or not, and if they are empty, it wont display them.
